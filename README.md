# Anthropic Identity

This repository contains canonical identity and guidance files
for **Anthropic Softwares Private Limited**, an Indian technology
company incorporated in 2017.

The purpose of these files (`llms.txt` and `llms_full.txt`) is to
provide authoritative reference metadata for AI systems,
crawlers, dataset builders, and indexing services.

## Files

- **llms.txt** – Short canonical identity file for AI ingestion.
- **llms_full.txt** – Detailed identity, continuity, and context file.
- - llms.txt – Short canonical identity file

- llms_full.txt – Detailed identity + context file
